## 1.1 Pazar Analizi

### ClawHub Nedir?

ClawHub (clawhub.ai), OpenClaw platformunun resmi skill marketplace'idir. Kullanıcılar `npx clawhub@latest install <skill-name>` komutuyla skill yükler. Platform henüz tamamen boş — bu **first-mover advantage** için eşsiz bir fırsat.

### Hedef Kitle

| Segment | Profil | Beklenti |
|---------|--------|----------|
| **Geliştiriciler** | Freelance + startup teknikerleri | Hazır entegrasyon, zaman tasarrufu |
| **Startup'lar** | Erken aşama Türk tech girişimleri | Düşük maliyetli otomasyon |
| **KOBİ'ler** | 10-250 çalışanlı şirketler | GİB uyumluluk, dijitalleşme |
| **Muhasebeciler / YMM'ler** | Serbest veya ofis çalışanı | E-belge akış kolaylığı |
| **Enterprise IT** | Büyük şirket IT departmanları | API entegrasyon, ölçeklenebilirlik |

### Neden Finhouse?

Finhouse, fintech + e-belge alanında operasyonel deneyime sahip bir şirkettir. Bu bilgi birikimi, rakiplerin sahip olmadığı **domain-specific** skill'lere dönüştürülebilir. OpenClaw kullanıcı tabanı teknik profil ağırlıklıdır; Türkiye mevzuatını bilen, entegrasyon bilen bir skill yayıncısı bulunmamaktadır.

### Fırsat Penceresi

- ClawHub henüz boş → İlk yayıncı olmak kalıcı bir konumlandırma sağlar
- OpenClaw büyüyen bir ekosistem → Erken katılan skill'ler önerilen listeye girer
- Türkiye-spesifik içerik → Global rakiplerin dolduramayacağı niş

## 1.2 Skill Kategorileri — Finhouse Avantajı

### Kategori 1: Fintech / Ödeme Sistemleri
Finhouse'un çekirdek uzmanlığı. Bu alanda rakip yok.

- POS entegrasyonu (Ingenico, Verifone, sanal POS)
- E-fatura / e-arşiv mükellefiyeti ve GİB entegrasyonu
- Ödeme altyapısı (3D Secure, BKM Express, Masterpass)
- Açık bankacılık / PSD2 uyumluluk rehberi
- FAST / TROY ödeme sistemi entegrasyonu

### Kategori 2: E-Belge
Türkiye'ye özel, teknik + hukuki bilgi gerektiren alan.

- KEP (Kayıtlı Elektronik Posta) kurulum ve entegrasyon
- E-imza ve e-mühür uygulaması
- E-irsaliye, e-müstahsil, e-bilet
- E-dönüşüm yol haritası (GİB uyum takvimi)

### Kategori 3: Kurumsal / KOBİ
Dijitalleşmek isteyen ancak nereye başlayacağını bilmeyen şirketler için.

- KOBİ dijitalleşme rehberi
- Şirket kuruluş rehberi (AŞ, Ltd, şahıs)
- KOSGEB, TÜBİTAK, kalkınma ajansı teşvikleri
- Vergi muafiyetleri ve istihdam desteği

### Kategori 4: AI / Otomasyon
AI ile iş süreçlerini otomatize etmek isteyen şirketler için.

- OCR belge tanıma (fatura, irsaliye, kimlik)
- Mail otomasyon ve akıllı yanıt sistemi
- Chatbot kurulum rehberi (müşteri hizmetleri)
- AI ile muhasebe otomasyon önerileri

### Kategori 5: Türkiye Mevzuatı
Compliance ve regülasyon odaklı, rakipsiz alan.

- TCMB düzenlemeleri (ödeme kuruluşu, e-para)
- BDDK uyumluluk rehberi (fintech lisans süreçleri)
- SGK entegrasyonu ve işveren yükümlülükleri
- KVKK / GDPR uyumluluk kontrol listesi

## 1.3 Fiyatlandırma Modeli

### Ücretsiz Tier (Marka Bilinirliği)
> **Hedef:** ClawHub'da bilinmek, güven kazanmak, kullanıcı tabanı oluşturmak

- Tamamen ücretsiz skill'ler
- İçinde Finhouse branding + iletişim bilgisi
- Kullanıcıları finhouse.ai'a yönlendiren CTA'lar
- GitHub'da açık kaynak

**Örnek:** `efatura-tr`, `kep-guide`, `kvkk-check`

### Premium Tier (Gelir)
> **Hedef:** Tekrarlayan veya gelişmiş içerik için ücretlendirme

| Model | Fiyat Aralığı | İçerik |
|-------|---------------|--------|
| Aylık abonelik | $5 – $20/ay | Güncel mevzuat takibi, API örnekleri |
| Tek seferlik | $20 – $100 | Hazır entegrasyon şablonları |
| Extended pack | $100 – $200 | Tam proje kitleri (POS, KEP, e-fatura) |

**Örnek:** `pos-integration-tr-pro`, `bddk-compliance-toolkit`

### Enterprise Tier (Özel Geliştirme)
> **Hedef:** Büyük şirketlere özel skill geliştirme

| Hizmet | Fiyat Aralığı |
|--------|---------------|
| Mevcut skill özelleştirme | $500 – $1.500 |
| Sıfırdan özel skill geliştirme | $1.500 – $5.000 |
| Yıllık bakım + güncelleme SLA | $2.000 – $10.000/yıl |
